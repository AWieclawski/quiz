Deploy war in the project directory: $ mvn clean package 
Run the application using: $ java -jar target/quiz*.war 
or do it by one command only in project dir.: $ mvn clean spring-boot:run 

and visit to: http://localhost:8080

Add some users via CLI commands ex.:
$ curl 'localhost:8080/user/add?userName=Andy&login=andy&password=AndyPassword&privilege=0&status=1'
//privilege established by APi is always EXPLOITER, never RULER
$ curl 'localhost:8080/user/add?userName=Betty&login=bety&password=BettyPassword&privilege=1&status=1'
$ curl 'localhost:8080/user/add?userName=Cindy&login=cin&password=CindyPassword&privilege=1&status=0'
$ curl 'localhost:8080/user/add?userName=Dolly&login=dol&password=DollyPassword&privilege=1&status=1'
$ curl 'localhost:8080/user/add?userName=Edward&login=eddy&password=EdwardPassword&privilege=1&status=1'

//acceptable example with wrong numbers (keys) in privilege and status fields ('11' instead of '1'):
curl 'localhost:8080/user/add?userName=Example&login=exy&password=ExamplePassword&privilege=11&status=11'

and list'em:
$ curl 'localhost:8080/user/all'
