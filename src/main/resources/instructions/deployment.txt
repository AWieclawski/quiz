Deploy war in the project directory: $ mvn clean package 
Run the application using: $ java -jar target/quiz*.war 
or do it by one command only in project dir.: $ mvn clean spring-boot:run 

and visit to: http://localhost:8080

Add some users via CLI commands ex.:
$ curl 'localhost:8080/user/add?userName=Andy&login=andy&password=AndyPassword&privilege=RULER&status=ACTIVE'
$ curl 'localhost:8080/user/add?userName=Bety&login=bet&password=BettyPassword&privilege=EXPLOITER&status=ACTIVE'
$ curl 'localhost:8080/user/add?userName=Cindy&login=cin&password=CindyPassword&privilege=EXPLOITER&status=EXCLUDED'

and list'em:
$ curl 'localhost:8080/user/all'
